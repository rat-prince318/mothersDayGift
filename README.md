# 为妈妈攀登祝福之塔

母亲节主题互动网页，通过攀登祝福之塔的方式向妈妈表达祝福。每一层楼对应一个独特的自然场景和一句祝福语，伴随第一人称爬塔动画切换楼层。

## 体验方式

1.直接打开https://rat-prince318.github.io/mothersDayGift/mothersDayGift.html
2.在浏览器中打开 `mothersDayGift.html`，点击「向上攀登」按钮逐层登塔。

## 视觉效果

- **程序化绘制**：所有场景元素均由 Canvas 实时绘制，无外部图片依赖
- **动态场景**：雪花飘落、海浪起伏、萤火虫游荡、花瓣飘舞等粒子动画
- **爬塔动画**：第一人称视角沿螺旋阶梯上行，模拟步伐节奏和视角转动
- **过渡效果**：楼层切换时柔和淡入淡出，配合祝福文字弹出

## 技术栈

纯前端 HTML + CSS + JavaScript（Canvas 2D），无需构建工具或运行时依赖。
